# EConnect

Sistema de administración escolar

## Características

- Gestión de alumnos
- Gestión de profesores
- Gestión de materias
- Gestión de cursos
- Base de datos SQLite
- Backend desarrollado con Flask
- Frontend HTML

---

## Tecnologías

- Python
- Flask
- SQLite
- HTML
- CSS

---

## Instalación

```bash
pip install Flask
python app.py
```

Luego abrir:

http://127.0.0.1:5000
